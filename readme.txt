How to run from Eclipse:
1. Load project from Open Projects from File System -> HoangLe_CSC464_Project1 -> Import all the nested projected.
2. Delete all jGrapht-related project from Project Explorer windows on the left hand side.
3. Build jGrapht library: Window -> Preferences -> expand Java -> select User Libraries -> New -> give it a name -> add JARs -> add all the jars in setting/jgrapht1.0.1/lib 0-> Apply and close
4. Project -> Properties -> Java Build Path -> switch to Libraries tab -> select the graph library you just defined -> Apply and close.
5. Select class named part1 -> run.
